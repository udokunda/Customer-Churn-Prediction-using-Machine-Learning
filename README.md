# Customer-Churn-Prediction-using-Machine-Learning
Customer churn analysis and prediction using Python, machine learning, feature engineering, model comparison, and business insights.

# Project Overview
Customer retention is a critical challenge for banks and subscription-based businesses. Acquiring new customers is often more expensive than retaining existing ones. This project uses machine learning techniques to identify customers who are likely to leave the bank (churn) and provides insights that can help improve customer retention strategies.
The objective of this project is to build predictive models capable of identifying customers at risk of churn and to uncover the key factors influencing customer departure.
# Business Problem
### Customer churn directly impacts revenue and long-term profitability. By accurately predicting which customers are likely to leave, businesses can:
- Reduce customer attrition
- Improve customer satisfaction
- Design targeted retention campaigns
- Increase customer lifetime value
- Make data-driven business decisions
# Project Workflow
## 1. Data Cleaning
- Removed irrelevant features (RowNumber, CustomerId, Surname)
- Checked for missing values and duplicates
## 2. Exploratory Data Analysis (EDA)
- Analyzed churn distribution
- Explored relationships between churn and customer demographics
- Created visualizations and correlation analysis
## 3. Feature Engineering
- Encoded categorical variables
- Scaled numerical features
## 4. Models Tested
- Logistic Regression
- Random Forest
- Gradient Boosting
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)
## 5. Model Evaluation
- Accuracy
- Precision
- Recall
- F1 Score

# Key Findings
The analysis revealed that customer churn is strongly influenced by:
- Customer age
- Account activity status
- Number of products owned
- Account balance
- Geographic location
  
Customers who are inactive or belong to specific demographic groups exhibit a higher probability of churn.
